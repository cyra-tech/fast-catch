# Fast Catch

Fast Catch is a lightweight macOS downloader with a clean interface and quick access from the menu bar.

---

## Overview

Fast Catch allows you to quickly download video and audio from supported platforms with a simple and fast workflow on macOS.

---

## Features

- Download video in **MP4**
- Download audio in **MP3**
- Quick access via **menu bar**
- Runs in background (does not quit on close)
- Simple and minimal UI
- macOS installer package (.pkg)

## Usage

1. Open the app  
2. Paste your video link  
3. Choose download location  
4. Select format (MP4 or MP3)  
5. Click **Download**
