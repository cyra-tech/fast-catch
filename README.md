# Fast Catch

Fast Catch is a lightweight macOS downloader with a clean UI and quick access from the menu bar.

---

## Overview

Fast Catch allows you to download videos and audio easily on macOS with a simple interface and fast workflow.

---

## Features

- Download videos in **MP4**
- Download audio in **MP3**
- Menu bar quick access
- Background running (does not quit on close)
- Simple and clean UI
- macOS installer support (.pkg)

---

## Usage

1. Open the app  
2. Paste your video link  
3. Choose download location  
4. Click **Download**
